---

### 🔹 Selection Sort
- Simple but inefficient  
- Time Complexity: **O(n²)**

---

### 🔹 Merge Sort
- Divide & conquer  
- Time Complexity: **O(n log n)**  
- Requires extra memory

---

### 🔹 Quick Sort
- Fast in practice ⚡  
- Time Complexity: **O(n log n)** (avg)

---

### 🔹 Heap Sort
- Uses heap structure  
- Time Complexity: **O(n log n)**

---

### 🔹 BST Sort
- Based on Binary Search Tree  
- Time Complexity: **O(n log n)**

---

# 🔍 Searching & Graph Traversal

### 🔸 Binary Search
- Efficient search on sorted arrays  
- ⏱ **O(log n)**

---

### 🔸 Breadth-First Search (BFS)
- Level-by-level traversal  
- Used in shortest path problems

---

### 🔸 Depth-First Search (DFS)
- Explores deep paths first  
- Useful for backtracking

---

### 🔸 Dijkstra / Uniform Cost
- Shortest path algorithm  
- ⏱ **O(E + V log V)**

---

### 🔸 A* Algorithm
- Heuristic-based search 🚀  
- Widely used in games & AI
